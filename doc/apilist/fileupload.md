#### API名称
---
```
文件上传
```

#### 描述
---
```
目前限制jpg,jpeg文件，请求必须包含一个path参数，表示要保存的相对路径
```

#### url
---
```
http://host:port/file/upload
```

##### 请求方法
---
```
POST
```

##### 调用样例
---
```

```

##### 输入参数
---
|字段     |类型     |备注
|---------|:------:|:-------|
|path     |string  |保存目录  |

##### 返回结果
---
*** JSON示例 ***
```
  {
  "meta": {
    "success": true,
    "message": "ok"
  },
  "data": [
    "/WEB-INF/file/1001/efedcd30-9aaf-4583-b2fe-581821718fac.jpg",
    "不支持的文件类型"
  ]
}
```

#### 说明
---
返回一个文件名列表，如果只上传了一个文件，列表中只有一个文件名，如果上传的文件类型不支持，列表里提示“不支持的文件类型”
